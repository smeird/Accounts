Backup & Restore

Create backups of your data or restore from an earlier snapshot. Use the tools below to download a copy of your information or load a previous backup.

Download Backup

Select which data to include in the compressed JSON backup file. User and account details are always included.

Need your transactions in another format? Visit the Exports page.

Restore Backup

Select a gzipped JSON backup file to restore any included transactions, categories, tags, groups, segments, projects, budgets, or settings.

How Data Is Organised

Segments contain categories, and each category contains canonical tags used to classify transactions. Tag aliases provide extra words or merchant descriptors that map back to those canonical tags (for example, an alias like “uber eats” mapping to a “Dining Out” tag). Groups remain separate from the segment/category/tag hierarchy and let you bundle transactions for projects, holidays, or any temporary initiative.
